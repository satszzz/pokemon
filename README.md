# 🎮 Pokémon Search App

A responsive Pokémon Search Application built using HTML, CSS, JavaScript, and PokéAPI. Users can search for Pokémon and instantly view details such as image, height, weight, and type.

🔗 **Live Demo:** https://satszzz-pokemon.netlify.app

👨‍💻 **Developed by:** Satheeswaran B

---

## ✨ Features

- 🔍 Search Pokémon by name
- 🖼️ Display Pokémon image
- 📏 Show Pokémon height
- ⚖️ Show Pokémon weight
- 🔥 Display Pokémon type
- ⚡ Real-time data fetching using PokéAPI
- 📱 Responsive design

---

## 🌐 API Used

**PokéAPI**

Endpoint:
https://pokeapi.co/api/v2/pokemon/

## 🚀 How to Run

1. Clone the repository
2. Open the project folder
3. Run `index.html`
4. Enter a Pokémon name and click **Search**
